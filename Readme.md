# IOT-backend

## Before starting, you need to download something...

- python(3.12.8)
- poetry
- mongoDB

## How to use

1. Create virtual environment

```
poetry env use python
```

2. Install dependency

```
poetry install
```

3. Add "poetry run" before your prompt
